# 项目介绍

实现虚拟登录查询QQ列表**设置你为特别关心的数量**

# 使用

如果您是一个编程语言的学习者，并且了解一定的语法，我推荐你：
```git
git clone https://github.com/WangRongsheng/eyeonu
```

如果你是一个普通使用者，我建议你下载[eyeonu.exe]()
